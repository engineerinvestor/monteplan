# Contributing to monteplan

Thank you for your interest in contributing! Please see our full
[Contributing Guide](https://engineerinvestor.github.io/monteplan/contributing/)
for details on:

- Setting up the development environment
- Running tests and type checks
- Code style and conventions
- Submitting pull requests

## Quick Start

```bash
git clone https://github.com/engineerinvestor/monteplan.git
cd monteplan
python -m venv .venv && source .venv/bin/activate
pip install -e ".[dev]"
pytest
```
